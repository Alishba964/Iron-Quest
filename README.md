#Iron Quest 
IronQuest is a C# Windows Forms application developed as a semester project.

## Features
- User Login
- Meal Management
- Progress Tracking
- Dashboard
- Database Connectivity

## Technologies Used
- C#
- Windows Forms
- Access (SQL)
- Visual Studio

## How to Run
1. Clone the repository.
2. Open the solution (.slnx/.sln) in Visual Studio.
3. Configure the MySQL database.
4. Build and run the project.
